## Notes
The Transmission client is using [Mullvad](https://mullvad.net/) as a VPN provider, feel free to fork this repo and change it.

## Instructions
- Create a .env file with the following variables:
    - *OPENVPN_PROVIDER*
    - *OPENVPN_CONFIG*
    - *OPENVPN_USERNAME*
    - *OPENVPN_PASSWORD*
- `docker compose up -d`
- Visit [localhost](http://localhost)
